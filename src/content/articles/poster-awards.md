<div style="font-family: Arial, sans-serif; font-size: 16px; line-height: 1.6; color: #333333; text-align: left;">
<p style="margin: 0 0 24px 0;">10% of the total poster award applications will be selected by the ICLTP Executive Committee and recognized at the banquet. The award consists of a certificate and a cash prize. The ICLTP gratefully acknowledges the important contributions students make to ICLTP.</p>
<div style="font-size: 18px; font-weight: bold; color: #0148a0; border-bottom: 2px solid #eaeaea; padding-bottom: 8px; margin-bottom: 16px;">Eligibility</div>
<p style="margin: 0 0 12px 0;">To be eligible for the award:</p>
<ul style="padding-left: 20px; margin: 0 0 24px 0;">
<li style="margin-bottom: 8px;">The student must present a poster as first author.</li>
<li style="margin-bottom: 0;">Register for the poster judging before the application deadline.</li>
</ul>
<div style="font-size: 18px; font-weight: bold; color: #0148a0; border-bottom: 2px solid #eaeaea; padding-bottom: 8px; margin-bottom: 16px;">Application Requirements</div>
<p style="margin: 0 0 12px 0;">The student application should contain:</p>
<ul style="padding-left: 20px; margin: 0 0 32px 0;">
<li style="margin-bottom: 8px;">The student's abstract.</li>
<li style="margin-bottom: 0;">The name of the advisor.</li>
</ul>
